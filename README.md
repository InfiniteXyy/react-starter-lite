# React Template Lite

## Main Stack
- Framework: `React 18`
- SPA: `React Router DOM`
- CSS: `Uno CSS` with tailwind style & dark-mode
- Lint: `ESLint` + `@vercel/style-guide`
- Language: `TypeScript`
- Format: `Prettier`
- Tools: `react-use`
- Icons: `iconify` `@uno/plugin-icons` (install @iconify-json/* when you need)
